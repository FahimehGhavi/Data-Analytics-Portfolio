# Extracting and Visualizing Stock Data

This project extracts Tesla and GameStop stock price and revenue data using Python, Yahoo Finance, and Web Scraping.  
The data is cleaned and visualized using Pandas and Plotly to show trends in historical share prices and revenue.

**Skills demonstrated:**  
- Data extraction (yFinance API, Web Scraping with BeautifulSoup)  
- Data cleaning and transformation (Pandas)  
- Data visualization (Plotly)
